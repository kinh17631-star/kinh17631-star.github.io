Official Installation Guide

1. Update your terminal environment

Termux (Android): apt

apt update && apt upgrade -y

iSH Shell (iOS): apk

apk update && apk upgrade
2. Install Git

Termux:

pkg install git -y
3. Clone and Launch PhoneInfo Pro

Run this final command to start the tool:

git clone https://github.com/kinh17631-star/MyToolHub && cd MyToolHub/phoneinfo && bash phoneinfo.sh
