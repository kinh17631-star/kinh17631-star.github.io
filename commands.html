Official Installation Guide

1. Update your terminal environment

For Termux Users Android

apt update && apt upgrade -y

For iSH Shell Users iOS

apk update && apk upgrade
2. Install Git (Required)

Install Git to clone the repository from GitHub.

Termux:

pkg install git -y

iSH Shell:

apk add git
3. Clone and Execute PhoneInfo Pro

Final command to launch the auditor.

git clone https://github.com/kinh17631-star/MyToolHub && cd MyToolHub/phoneinfo && bash phoneinfo.sh
