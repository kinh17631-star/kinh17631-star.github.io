🚀 MyToolHub Commands

FOR TERMUX COMMANDS (IP-TRACER)

1. Update your Termux terminal

pkg update && pkg upgrade -y

2. Install Python in Termux

pkg install python git -y

3. Clone IP-Tracer tool

git clone https://github.com/kinh17631-star/MyToolHub

4. Enter Tool Directory

cd MyToolHub/IP-Tracer

5. Run Installer & Tool

python3 install.py && python3 main.py
FOR ISH SHELL COMMANDS (IP-TRACER)

1. Update iSH Packages

apk update && apk upgrade

2. Install Python & Git in iSH

apk add python3 git

3. Clone Repository

git clone https://github.com/kinh17631-star/MyToolHub

4. Launch IP-Tracer

cd MyToolHub/IP-Tracer && python3 install.py && python3 main.py
