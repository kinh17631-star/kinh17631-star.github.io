🚀 MyToolHub Commands

FOR TERMUX COMMANDS

1. Update your Termux terminal

pkg update && pkg upgrade -y

2. Install Python in Termux

pkg install python git -y

ADVERTISEMENT

3. Clone IP-Tracer tool

git clone https://github.com/kinh17631-star/MyToolHub

4. Enter Tool Directory

cd MyToolHub/IP-Tracer

ADVERTISEMENT

5. Run Installer & Tool

python3 install.py && python3 main.py
