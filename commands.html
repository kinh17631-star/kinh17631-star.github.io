🚀 MyToolHub Commands

FOR TERMUX COMMANDS

1. Update & Upgrade Terminal

pkg update && pkg upgrade -y

2. Install Python & Git

pkg install python git -y

3. Clone IP-Tracer tool

git clone https://github.com/kinh17631-star/MyToolHub

4. Enter Tool Directory

cd MyToolHub/IP-Tracer

5. Start Tool

python3 install.py && python3 main.py
FOR ISH SHELL

1. Run All-in-One Command

apk update && apk add python3 git && git clone https://github.com/kinh17631-star/MyToolHub && cd MyToolHub/IP-Tracer && python3 install.py && python3 main.py
