Part 4: IP-Tracer Tool

1. Update and upgrade your terminal

pkg update && pkg upgrade -y

2. Install Python language

pkg install python -y

3. Clone the tool from GitHub

git clone https://github.com/kinh17631-star/MyToolHub

4. Go to IP-Tracer directory

cd MyToolHub/IP-Tracer

5. Run the tool installer and main file

python3 install.py && python3 main.py
